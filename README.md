# Description
Small Alpine-based Docker image with installed cli tools: `awscli`, `kubectl` (latest, and older _running_ version), `helm` and `terraform`.


# Versions

* awscli     `1.16.131`
* helm     `v2.13.0`
* kubectl (latest)   `v1.13.4` => _kubectl_latest_
* kubectl (running)  `v1.11.6` => _kubectl_
* Terraform     `0.11.11`
